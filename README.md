# HOAXAGON

Hoaxagon es un juego de puzles arcade en el que eres contratado por el enigmático ICOSAMUEL para moderar las publicaciones de una red social poblada por polígonos.
A lo largo de una partida, deberás identificar las falacias argumentativas presentes en cada comentario a evaluar, y recibirás una puntuación según tu desempeño.
Existe un modo entrenamiento para practicar con falacias particulares y un modo arcade en el que se añadirán falacias gradualmente.
